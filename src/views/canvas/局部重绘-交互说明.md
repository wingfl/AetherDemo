# 局部重绘 - 交互说明文档

## 一、弹窗概述

点击图片卡片的"局部绘制"按钮后，弹出模态遮罩弹窗，标题为"局部重绘"。弹窗从上到下分为四个区域：标题栏、画布区域、画笔尺寸条、底部工具栏（两行）。

退出方式：点击右上角 X 按钮 / 点击弹窗外遮罩区域 / 按 ESC 键。退出时直接关闭弹窗，不保存任何编辑状态。

---

## 二、标题栏

"局部重绘" 文字 + 右上角关闭按钮（X 图标）。

---

## 三、画布区域

深色背景（#0a0a0a），图片居中显示，使用 Konva 渲染。

### 3.1 缩放

| 操作 | 行为 |
|------|------|
| 鼠标滚轮 | 以鼠标指针位置为中心缩放 |
| 底部 `-` / `+` 按钮 | 以画布中心为基准缩放，每次步进 25% |
| 缩放范围 | 25% ~ 300%，缩放比例实时显示 |

### 3.2 光标

| 工具 | 光标状态 |
|------|----------|
| 涂抹 | 图片区域内：隐藏系统光标，显示画笔圆圈（绿色实心，半径 = 画笔尺寸 × 缩放倍率）；图片区域外：恢复默认光标 |
| 橡皮擦 | 图片区域内：隐藏系统光标，显示橡皮擦圆圈（红色实心）；图片区域外：恢复默认光标 |
| 框选 | 十字光标（crosshair） |
| 移动 | 手型光标（grab），拖拽中变为握紧（grabbing） |

---

## 四、画笔尺寸条

位于画布区域正下方，仅在选中 **涂抹** 或 **橡皮擦** 工具时可见。

切换到移动/框选工具时，尺寸条隐藏但**保持占位高度**（42px），防止页面跳动。

| 元素 | 说明 |
|------|------|
| "画笔尺寸" 标签 | 固定文字 |
| `-` 按钮 | 画笔尺寸减 1，最小值为 1 |
| 滑块 | range 输入，拖拽调节，范围 1 ~ 100 |
| `+` 按钮 | 画笔尺寸加 1，最大值为 100 |
| 数值显示 | 实时显示当前画笔尺寸（如 10） |

默认值：10px，可调范围：1px ~ 100px，涂抹和橡皮擦共用同一个尺寸参数。

---

## 五、底部工具栏 - 第一行

从左到右分为四个区段，中间用竖线分隔。

### 5.1 工具选择区

四个工具按钮，同一时间只能激活一个：

| 工具 | 图标 | 说明 |
|------|------|------|
| 移动 | 四向箭头 | 拖拽图片在画布容器内平移 |
| 框选 | 四宫格 | 在图片上拖拽矩形区域，松手后自动填充为涂抹内容 |
| 涂抹 | 画笔 | 手动涂抹需要重绘的区域（默认选中） |
| 橡皮擦 | 橡皮 | 擦除已涂抹的区域，**无涂抹内容时禁用** |

**各工具交互细节：**

**涂抹工具**
1. 选中涂抹工具（默认选中）
2. 鼠标移入图片区域，显示绿色圆圈光标，圆圈大小 = 画笔尺寸 × 缩放倍率
3. 在图片区域内按住鼠标左键并拖动进行涂抹
4. 涂抹区域以绿色 40% 透明度实时显示覆盖在原图上
5. 松开鼠标完成一笔，自动记录历史
6. 只能在图片区域内涂抹，超出图片边界不绘制
7. 多次涂抹会叠加

**橡皮擦工具**
1. 点击橡皮擦按钮（需已有涂抹内容，否则按钮禁用）
2. 鼠标移入图片区域，显示红色圆圈光标
3. 在已涂抹区域内拖动进行擦除
4. 被擦除的区域会彻底移除涂抹（不留残留）
5. 松开鼠标完成擦除，自动记录历史

**框选工具**
1. 选中框选工具，光标变为十字
2. 在图片上按住鼠标左键拖拽，绘制虚线矩形选区
3. 选区实时显示，带绿色虚线边框 + 浅绿色填充
4. 松开鼠标后，选区自动填充为涂抹内容（等同于一次性涂抹整个矩形区域）
5. 自动记录历史
6. 拖拽距离过小（宽或高 < 5px）时不生效，视为误触

**移动工具**
1. 选中移动工具，光标变为手型
2. 在画布区域按住鼠标左键拖拽
3. 图片跟随鼠标移动（拖拽中光标变为握紧状态）
4. 松开鼠标停止移动
5. 移动操作不会改变涂抹内容，也不记录历史

### 5.2 操作区

三个图标按钮：

| 按钮 | 图标 | 说明 |
|------|------|------|
| 撤销 | 左弯箭头 | 撤销上一步涂抹操作，退回到上一个历史状态。无更多历史时禁用 |
| 重做 | 右弯箭头 | 重做被撤销的操作。无更多可重做时禁用 |
| 清除 | 垃圾桶 | 清除所有涂抹内容。无涂抹内容时禁用 |

**历史记录规则：**
- 最多保存 50 步历史，超出后自动丢弃最早的记录
- 涂抹、橡皮擦、框选填充、清除操作都会记录一个历史快照
- 撤销/重做不会产生新的历史记录
- 撤销操作后若涂抹内容变为空且当前是橡皮擦，自动切回涂抹工具

### 5.3 缩放区

| 元素 | 说明 |
|------|------|
| `-` 按钮 | 缩小 25% |
| 缩放比例文本 | 显示当前缩放百分比（如 100%） |
| `+` 按钮 | 放大 25% |

## 六、底部工具栏 - 第二行

从左到右依次为：

| 元素 | 类型 | 说明 |
|------|------|------|
| 提示词输入框 | textarea | 输入重绘区域的描述文本。placeholder："请描述你的重绘区域内容，不填则基于子原图生成" |
| 模型下拉 | select | 选择 AI 生成模型 |
| 张数下拉 | select | 选择生成张数（1/2/3/4 张） |
| 开始重绘 | button | 提交生成请求，**无涂抹内容时禁用** |

### 6.1 提示词输入框

- 自动高度：最少 1 行（38px），最多 3 行（84px）
- 超出 3 行后内部滚动，滚动条隐藏
- 不填则基于子原图生成

### 6.2 模型选项

| 选项 | 值 |
|------|----|
| Stable Diffusion XL | `stable-diffusion-xl`（默认） |
| Stable Diffusion 3 | `stable-diffusion-3` |
| SD3 Turbo | `stable-diffusion-3-turbo` |
| Flux.1 Dev | `flux-1-dev` |
| Flux.1 Schnell | `flux-1-schnell` |

### 6.3 生成张数

默认 4 张，可选 1 / 2 / 3 / 4 张。

---

## 七、边界约束

| 场景 | 行为 |
|------|------|
| 大图自动缩放 | 上传图片超过 800x600 时，按比例缩放到画布适配尺寸 |
| 绘制区域限制 | 涂抹和橡皮擦只能在图片矩形内操作，超出不响应 |
| 历史上限 | 最多 50 步，超出后自动丢弃最早的历史 |
| 弹窗内缩放范围 | 25% ~ 300% |
| 画笔尺寸范围 | 1px ~ 100px |
| 框选最小面积 | 宽或高 < 5px 时视为误触，不生成涂抹 |
| 多图片独立 | 每张图片的涂抹数据独立保存，互不影响 |
| 窗口 resize | 通过 ResizeObserver 监听，画布尺寸自适应容器变化 |
| 禁用联动 | 橡皮擦、清除、开始重绘三个操作均受"是否有涂抹内容"控制，无内容时禁用 |
| 自动切回画笔 | 清除操作、撤销后内容为空时，若当前是橡皮擦则自动切回涂抹工具 |
